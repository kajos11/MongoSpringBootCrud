Contacts CRUD OPERATION using Spring Boot, Embedded H2 Mongo DB 

Steps to run:
1. Maven install in project root folder: 
	mvn install
2. Run the application using command:
	mvn spring-boot:run
3. From PostMan or any REST client, we can access the api's: ex: http://localhost:8080/contacts


Embedded MongoDb details:
URL: mongodb://localhost:{PORT}/embeded_db
NO CREDENTIALS
